# Professional Medical Visual Standards

## UI Philosophy
- **Style**: "Clean, Clinical, Confidence-Inspiring".
- **Primary Goal**: Clarity and ease of research. Compliance data should be scanable and unambiguous.
- **Accessibility**: AA/AAA WCAG compliance is mandatory due to the technical/medical nature of the content.

## Typography
- **Primary Font**: `Inter` or `Roboto` (Clean, modern sans-serif).
- **Secondary Font**: `Outfit` (for headings/hero sections).
- **Rule**: Minimum 16px font size for body text to ensure readability of complex legal/technical text.

## Color Palette
- **Primary (Trust)**: `#1a5f7a` (Deep Teal) / `#00a8cc` (Medical Blue).
- **Success**: `#2ecc71` (Emerald) - Indicates "Compliant" or "Passed".
- **Warning**: `#f39c12` (Amber) - Indicates "Action Required" or "Gap Identified".
- **Danger**: `#e74c3c` (Alizarin) - Indicates "Non-Compliant" or "Critical Failure".
- **Background**: `#f8f9fa` (Light Grey) or Clean White.

## Layout & Components
- **Dashboards**: Use whitespace to separate complex compliance modules.
- **Checklists**: Clear, large checkboxes with animated "Success" states.
- **Document Preview**: Use a clean, paper-like shadow effect for technical file previews.
- **Tables**: Strictly bordered with alternating row colors (Zebra striping) for readability of dense regulatory data.
- **Icons**: Standard `strokeWidth` is **2.5** to ensure a premium, clinical feel. Icons should typically match the color of their adjacent text (e.g., Primary Teal for normal labels).
- **Buttons**: Every button must have a `gap: 0.5rem` (8px) between icon and text. Use sizing variants (`primarySm`, `primaryLg`) to match typography.

## Interactive Elements
- **Feedback**: Use the global `Toast` system for all state transitions (e.g., "Technical File Saved").
- **Admin**: Gemma-driven content editing should have a "Review & Publish" workflow to ensure accuracy.

## Imagery
- **Source**: AI-generated medical/office environment imagery.
- **Style**: Professional, brightly lit, high-resolution.
- **Tags**: `Professional medical office, modern laboratory, high-tech regulation interface, clinical, clean, 4k`.
